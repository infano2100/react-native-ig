# react-native-ig

## Getting Started

Download with: 

```sh 
git clone https://github.com/infano2100/react-native-ig.git && cd react-native-ig
```

Install Libs:

```sh 
- npm i
- cd ios && pod install
```

Run with:

```sh 
- run ios => react-native run-ios
- run android => react-native run-android
```
